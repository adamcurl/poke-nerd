# PokeNerd
A web app to pull fun stats for Pokemon using [PokeAPI](https://pokeapi.co/)

### Tech Stack
- **Base:** [Vite](https://vitejs.dev/guide) + [React](https://react.dev/learn) + [TS](https://www.typescriptlang.org/docs)
- **Queries + State Management:** [TanStack Query](https://tanstack.com/query/v5/docs/react/overview)
- **Type Generation:** [Swagger-TypeScript-Api](https://github.com/acacode/swagger-typescript-api)
